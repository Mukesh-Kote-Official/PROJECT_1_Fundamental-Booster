# PROJECT 1 – Fundamental Booster

## Overview

This Excel project is designed to practice and improve fundamental data handling, calculation, and analysis skills using Microsoft Excel.

The workbook contains multiple datasets related to:

* Student Grades
* Sales Data
* Employee Data

The project helps in understanding:

* Basic Excel formulas
* Data analysis techniques
* Formatting and calculations
* Report preparation

---

# Workbook Structure

## 1. Project Instructions Sheet

This sheet contains the instructions and objectives of the project.

### Purpose

* Understand project requirements
* Learn expected outputs
* Follow practice tasks step-by-step

---

## 2. Students Grade Sheet

This sheet contains student academic records.

### Columns Included

* Student ID
* Student Name
* Math Marks
* Science Marks
* English Marks
* Total Marks
* Average Marks
* Grade

### Excel Concepts Used

* `SUM()` → Calculate total marks
* `AVERAGE()` → Calculate average marks
* `IF()` → Assign grades
* `ROUND()` → Round average values

### Example Formulas

```excel
=SUM(C2:E2)
```

```excel
=AVERAGE(C2:E2)
```

```excel
=IF(G2>=80,"A",IF(G2>=60,"B","C"))
```
